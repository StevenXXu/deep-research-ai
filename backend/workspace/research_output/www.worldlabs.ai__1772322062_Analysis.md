## 2. Product Deep Dive

World Labs' primary product is Marble, a multimodal world model that generates spatially consistent, high-fidelity 3D worlds [1]. It allows users to create explorable 3D environments from text, images, videos, or 360 panoramas [1].

**Features:**

*   **Multimodal Inputs:** Supports text, images, videos, and 360 panoramas as input [1].
*   **3D Layout Control:** Enables precise control over the 3D layout of generated worlds [1].
*   **Interactive Editing:** Allows users to tweak specific elements or reshape entire 3D worlds [1].
*   **World Expansion and Combination:** Facilitates expanding, editing, and combining generated worlds to build larger, more immersive environments [1].
*   **Versatile Outputs:** Supports downloading and exporting in various 2D and 3D formats [1].
*   **Spatial Consistency:** Generates spatially cohesive, high-fidelity, and persistent 3D worlds [2].
*   **World API:** A public API for generating explorable 3D worlds from text, images, and video, bringing Marble's world modeling capabilities into applications [2].

*   **3D Graphics Engines:** Custom or open-source rendering engines for generating and displaying 3D environments.
*   **Cloud Computing Platforms:** AWS, Azure, or Google Cloud for model training, data storage, and API deployment.

Given AMD, NVIDIA, and Autodesk are investors, World Labs likely uses their hardware and software such as NVIDIA GPUs and Autodesk's 3D CAD software [2], [8].
